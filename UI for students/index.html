Приклади UI Елементів

1. Перемикач (Radiobutton)

Дозволяє обрати лише один варіант із кількох.

2. Прапорець (Checkbox)

Дозволяє обирати кілька варіантів одночасно.

3. Текстове поле

Поле для введення тексту.

4. Вкладки (Tabs)

Використовуються для перемикання між різними розділами.

5. Кнопка

Виконує дію при натисканні.

6. Текстова мітка (Label)

Відображає описовий текст поруч із полем вводу.

7. Посилання (Link)

Елемент навігації, що веде на іншу сторінку.

Перейти до Google

8. Підказка (Tooltip)

Показує додаткову інформацію при наведенні.

Наведи на мене Це приклад підказки

9. Випадаючий список

Дозволяє обрати один варіант зі списку.

10. Таблиця даних (Data grid)
